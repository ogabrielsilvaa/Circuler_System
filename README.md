# Circuler ♻️📚

**Circuler** é uma plataforma digital focada na economia circular de livros. O sistema conecta cidadãos que possuem livros ociosos a estudantes que precisam de materiais didáticos e literários, utilizando "Pontos de Coleta" parceiros para garantir uma distribuição segura e descentralizada.

Este projeto é um MVP (Produto Mínimo Viável) Fullstack com foco em **Direitos Humanos e Diversidade Cultural**, atendendo diretamente aos ODS 4, 10, 12 e 16 da ONU.

---

## 📱 Visão Geral da Arquitetura

O projeto adota uma arquitetura cliente-servidor e está estruturado como um *monorepo*, contendo o aplicativo móvel (frontend) e a API (backend) no mesmo repositório para facilitar a avaliação.
